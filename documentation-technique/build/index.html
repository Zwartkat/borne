    Build et Compilation - Guide Complet¶

    Guide pour compiler la Borne d'Arcade.

    🔨 Compilation Basique¶

    Avec le Script¶

    # Linux / Mac
bash compilation.sh

# Windows
compilation.bat

    Ce script compile: 1. Le module Arcade 2. Le module Main 3. Tous les jeux individuels

    Manuellement¶

    # Compiler Arcade
cd Arcade
javac -cp .;../MG2D.jar *.java

# Compiler Main
cd ..
javac -cp .;./MG2D.jar *.java

# Compiler un jeu
cd projet/Columns
javac -cp .;../..;../../MG2D.jar *.java

    🏗️ Avec Maven (Optionnel)¶

    mvn clean compile
mvn package

    ℹ️ Commandes Disponibles¶

    • compilation.sh - Compile tout
    • clean.sh - Nettoie les fichiers compilés
    • ./launch.sh - Lance la borne
