# Installation pour Développeurs

Guide pour mettre en place votre environnement de développement.

## 📋 Prérequis

- Java Development Kit (JDK) 17+
- Maven 3.8+ (optionnel)
- Git
- Un éditeur (VSCode, IntelliJ, etc.)

Voir [setup.md](setup.md) pour les détails.

## 🚀 Démarrage Rapide

```bash
# 1. Cloner
git clone https://github.com/Zwartkat/borne.git
cd borne

# 2. Configurer
source installation-dev/setup.sh  # ou .bat pour Windows

# 3. Compiler
bash compilation.sh

# 4. Tester
./launch.sh
```
